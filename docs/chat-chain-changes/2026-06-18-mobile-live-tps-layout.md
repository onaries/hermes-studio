# Mobile live TPS layout

- Lets the composer top bar wrap on mobile so the live TPS/context meter gets its own row instead of being squeezed out by controls.
- Splits context usage and live TPS into separate inline spans so long context text ellipsizes while TPS remains visible.
- Keeps the context progress bar compact and preserves desktop layout behavior.
