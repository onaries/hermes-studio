feature: Mobile artifacts browsing
impact: Mobile Artifacts drawer now shows the file list first and opens one artifact at a time in a compact detail view.

Changes the mobile Artifacts drawer from a stacked list-plus-preview layout to a list-to-detail flow. On phones the artifact list uses the full drawer height; tapping an artifact opens a single-file viewer with a compact toolbar and a back-to-list control. Desktop keeps the existing two-column list and preview layout.
