# Collapsible session agent groups

The Chat sidebar agent groups introduced for Codex, Claude Code, and Hermes now behave as collapsible headers. Clicking a group header toggles its sessions, updates the chevron, and persists the collapsed groups in localStorage so the sidebar keeps the user's preferred compact layout across reloads.
