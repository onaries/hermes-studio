从通电到对话

第一次使用需要配网、连接 Hermes Studio，并在电脑端完成语音和模型服务配置。

设备配网

  1. 用 USB 给小方盒供电，内置电池。
  2. 首次开机进入配网模式。
  3. 手机或电脑连接 Hstudio-WIFI。
  4. 进入 192.168.4.1，选择 Wi-Fi 进行连接并保存。
  5. 等待页面跳转，会显示当前设备在当前局域网的 IP。
  6. 点击下方设备登录，使用 Hermes Studio 账号密码登录，并选择对应 profile。
http://192.168.4.1

连接 Hermes Studio

  1. 电脑和 ESP32 保持在同一个局域网。
  2. 打开电脑端 Hermes Studio，进入设置页面的语音设置。
  3. 完成语音 API Key 的设置。
  4. 浏览器进入当前设备的局域网 IP 设置页。
  5. 在网络设备区域点击刷新探测。
  6. 服务端填写 Hermes Studio 的服务端 IP + 端口。
  7. 也可以点击域名添加，保存后完成连接。
示例：http://电脑局域网IP:8748

常见问题

这些问题适合直接放到商品页，能提前过滤不适合买家。

它能脱离电脑单独聊天吗？

不能。它需要连接电脑端或者云服务器 Hermes Studio。ESP32 负责硬件交互，AI 能力由 Hermes Studio 和用户配置的服务提供。

买来后需要配置什么？

需要给设备配 Wi-Fi，在 Hermes Studio 中完成配对，并配置语音识别、语音合成和模型服务。

为什么不能填写 127.0.0.1？

对 ESP32 来说，127.0.0.1 代表 ESP32 自己，不是你的电脑。需要填写电脑在局域网里的 IP 地址。

声音适合什么场景？

适合桌面近距离听，定位是语音提示和近场对话反馈，不是大功率音箱或远场智能音箱。

可以二次开发吗？

可以。它适合有 ESP32、Arduino、PlatformIO 或 Hermes Studio 使用基础的用户继续修改固件和交互逻辑。
